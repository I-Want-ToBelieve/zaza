## Usage

Backup
```zsh
mv ~/.config/nvim ~/.config/nvim.bak
```

Clone Repo
```zsh
git clone https://github.com/backtolife2021/zaza.git ~/.config/nvim
cd ~/.config/nvim
```

Install dependencies
```zsh
pnpm install
```

Build and init
```zsh
pnpm run one:clean
```

Open File
```zsh
nvim ~/.config/nvim/src/index.ts
```
